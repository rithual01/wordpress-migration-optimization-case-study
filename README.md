# wordpress-migration-optimization-case-study
Diseñé e instalé desde cero un servidor VPS con HestiaCP (Nginx/Apache) para el portal de investigación de la UNI. Audité y depuré una base de datos histórica (2015-2026), eliminando 11 años de datos basura. Resolví con WP-CLI e ingeniería inversa un fallo crítico de SQL, recuperando intactas 66 tablas de registros y optimicé medios a WebP.
